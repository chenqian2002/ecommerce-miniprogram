- **配色方案**：暖色系（橙红 #ff6b35 → #ff8c42）
- **布局风格**：参考 KFC 电商应用
- **交互反馈**：点击缩放、阴影变化、色彩高亮

#### 🔧 技术栈

**后端**
- 框架：FastAPI
- ORM：SQLAlchemy
- 数据库：SQLite
- 认证：JWT
- 加密：SHA256

**前端**
- 技术：WeChat MiniProgram
- 标记语言：WXML
- 样式：WXSS
- 脚本：JavaScript (ES6)
- 存储：本地存储 (wx.getStorageSync)

#### 🔐 认证流程

1. 登录页输入手机号+密码
2. 后端验证密码（SHA256）
3. 返回 JWT token
4. 前端保存 token 到本地存储
5. 后续请求自动添加 Authorization header
